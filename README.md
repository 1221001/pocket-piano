# Pocket Piano Web

A mobile-friendly piano web app that works in Safari on iPhone.

- White keys show subtle labels such as `D1`
- The settings panel lets you change white key width
- Swipe horizontally to browse the full keyboard
- Tap the sound button once, then play with touch

## Files

- `index.html`
- `styles.css`
- `script.js`

## How to use on iPhone

1. Put these files on any static web host.
2. Open the page in Safari on iPhone.
3. Tap `Tap to enable sound`.
4. Swipe sideways to see more keys.
5. Use the slider to change key size.

You can also open the page locally on a computer browser first to test the layout.
